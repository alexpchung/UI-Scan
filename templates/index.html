This is Code Scan App

Use for scanning and searching the source files for Fusion Application Labels.

All the attribute values are resolved using their respective resource bundles. If the bundle library is not found, the bundle name will appear in the missing bundle file inside the output folder.

The data library is provided by Search Source Tool(http://adc2100106.us.oracle.com/cgi-bin/searchSource) and this application use Search Source Tool as an API.

The search results output files are stored in the application subfolder named: output/[dialog|explore|icon]***.txt
